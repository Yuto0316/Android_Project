# Android_Project
Project Outline: UniTrade - Student-Exclusive Marketplace Platform

1. Project Description
UniTrade is a mobile application designed to provide a safe and reliable environment for university students to buy, sell, or exchange items. By restricting the community to students, it eliminates the need to interact with the general public and focuses on circulating essentials such as textbooks, furniture, electronics, and specialized equipment within a trusted academic network.

2. Problem Addressing
Existing broad-market platforms (such as Facebook Marketplace or Craigslist) present several challenges for students:
・Safety Concerns: Psychological and physical barriers to meeting strangers from the general public.

・Shipping Costs: The high cost of shipping bulky furniture or low-priced textbooks often outweighs the profit.

・Waste Generation: Useful items are frequently discarded during graduation or move-out seasons due to lack of time or a convenient way to sell locally.

・Business Importance: This project addresses these issues by creating a niche secondary market tailored to the unique "short-term ownership and disposal" cycle of student life.

3. Platform
Android OS: Developed specifically for Android to leverage mobile mobility, allowing users to list items and communicate instantly on the go.

4. Front/Back end Support
Frontend: Java (Android Studio) will be used to build a robust and standard development environment.

Backend/Database: Firebase will be utilized to seamlessly integrate real-time chat, user authentication, and cloud storage for product images.

5. Functionality
・University Domain Authentication: Registration is restricted to users with verified university email addresses (.edu, etc.) to ensure community trust.

・Smart Listing: Users can easily list items by taking a photo and selecting a category (Textbooks, Electronics, Dorm Essentials, etc.).

・Location & Campus Filtering: A search function that allows users to filter items by their own university or nearby campuses.

・Secure In-App Messaging: A built-in messaging system to negotiate prices and arrange meet-up locations on campus without using external apps.

6. Design (Wireframes)
・Home Screen: A feed-style interface displaying new arrivals categorized by university with photos and prices.

・Item Detail Screen: Displays item condition, price, seller ratings, and a "Chat to Inquire" button.

・My Page: Allows users to manage active listings, view transaction history, and access their "saved" items list.
